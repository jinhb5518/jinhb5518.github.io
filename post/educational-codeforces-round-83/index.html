Educational Codeforces Round 83

D

题意：计算满足下述序列的数量
1.每个数组包含nnn个元素；
2.每个元素都是111到mmm之间的整数；
3.对于每个数组，只有一对相等的元素；
3.对于每个数组aaa，都有一个索引i，使得数组在第i个元素之前严格升序，在其第i个元素之后严格降序（正式而言，这意味着aj<aj+1a_j<a_{j+1}aj​<aj+1​，如果j<ij <ij<i，而aj>aj+1a_j> a_{j + 1}aj​>aj+1​，如果j≥ij\ge ij≥i）。
题解：因为一个数组里有n−1n-1n−1个元素,故需要从mmm个元素中选出n−1n-1n−1个，其次枚举那个重复的元素，因为最大的元素必不能是重复的元素，故有n−2n-2n−2种可能，最后考虑其余的元素应该放在最大元素的两边，故有2n−32^{n-3}2n−3种可能，所以最终答案为(n−2)Cmn−12n−3\left(n-2\right)C_m^{n-1}2^{n-3}(n−2)Cmn−1​2n−3
代码：

#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
const ll mod=998244353;
ll qpow(ll a,ll b){
	ll ans=1;
	while(b){
		if(b&1)ans=(ans*a)%mod;a=(a*a)%mod;b>>=1;
	}return ans;
}
ll n,m,r=1,ans=1;
int main()
{
	
	scanf("%lld%lld",&n,&m);
	if(n==2){
		puts("0");return 0;
	}
	for(ll i=n;i<=m;i++)ans=(ans*i)%mod;
	for(ll i=1;i<=m-n+1;i++)r=(r*i)%mod;
	ans=(ans*qpow(r,mod-2))%mod;
	ans=(ans*(n-2))%mod;
	ans=(ans*qpow(2,n-3))%mod;
	printf("%lld\n",ans);
	
}
